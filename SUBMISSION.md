# Submission Instructions

## How to Submit

1. Fork this repository
2. Create a new branch:
   feature/"your-name"
3. Implement the solution in your fork
4. Open a Pull Request to the `main` branch of this repository

---

## Pull Request Requirements

Your PR must include:
- A complete Xcode project
- A README in the project root explaining:
  - How to run the app
  - Architectural decisions
  - Memory management approach
  - Assumptions made
  - Time spent on the task

---

## Notes
- Do not squash commits
- Commit history should reflect your development process
